회색리 연속 실종 사건
// 실종 현황

최근 3개월간 회색리에서 주민 3명이 흔적도 없이 사라졌습니다.
모두 동네 북쪽 골목 인근 거주자였습니다.

// 목격자 진술

여러 주민들이 밤마다 옆집 — 한윤재(35세)의 집에서
이상한 소리가 들린다고 증언했습니다. 경찰은 수사를 중단했습니다.

// 발견된 단서

실종자 중 한 명의 가방이 한윤재 집 담장 근처에서 발견됐습니다.
그 안에는 당신 가족의 사진이 들어있었습니다.
